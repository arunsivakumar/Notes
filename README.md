# NotesApp
iOS - Notes App Clone, with additional features (using Core data)


## Description

IOS notes app clone with tags, colored categories and search.


## IOS Requirements
* XCODE-8
* Swift 3.0

## Status (In-Progress)

1. Colors for Categories
2. Search
